# Clube USA: Projeto de Automação de Ofertas

Bem-vindo ao Clube USA! Este projeto é um sistema automatizado para raspar (scrape), processar e postar ofertas de produtos em canais de mídia social (Telegram e WhatsApp), com foco inicial em ofertas dos EUA. Ele inclui integração com APIs de terceiros (Amazon Associates, Telegram Bot API, Twilio WhatsApp
