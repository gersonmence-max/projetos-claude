BRASILDEALS - PROJETO FUNCIONAL E PRONTO PARA PRODUÇÃO
================================================================================

STATUS: ✓ COMPLETO E OPERACIONAL

Este é um projeto Python production-ready que automatiza a distribuição de 
ofertas de produtos para Telegram e WhatsApp com integração real de APIs.

================================================================================
CARACTERÍSTICAS PRINCIPAIS
================================================================================

SCRAPING:
- Fetch REAL de RSS feeds (SlickDeals ou custom)
- Parsing inteligente de titles e descrições
- Detecção automática de categorias
- Extração de discounts percentuais
- Deduplicação de deals no banco

PROCESSAMENTO:
- Filtering por discount mínimo (configurável)
- Filtering por categorias relevantes
- Formatação de mensagens em HTML e Markdown
- Geração de links de afiliado Amazon
- Status tracking de cada post

DISTRIBUIÇÃO:
- Telegram: Integração REAL via python-telegram-bot (async/sync)
  * Suporta channels, groups, direct messages
  * Retry automático em caso de falha
- WhatsApp: Integração REAL via Twilio
  * SMS alternativo incluído
  * Suporta lista de múltiplos destinatários
  * Rate limiting configurável

AGENDAMENTO:
- APScheduler com cron triggers
- 4x diárias em horários configuráveis (padrão: 6, 10, 14, 18 UTC)
- Retry automático em falhas consecutivas
- Alertas de erro após 2 falhas

BANCO DE DADOS:
- PostgreSQL com SQLAlchemy ORM
- 4 tabelas principais: Deals, Channels, PostLogs, Commissions
- Índices otimizados para queries
- Pool de conexão com reciclagem automática

LOGGING:
- JSON format para fácil parsing
- Console + Arquivo simultâneos
- Rotating file handler (10MB + 10 backups)
- Configurable log levels

================================================================================
COMO USAR
================================================================================

INSTALAÇÃO:

1. Clone/extraia o projeto
2. Execute: bash setup.sh
3. Configure .env com suas credenciais
4. Execute: python main.py --mode health

MODOS DE OPERAÇÃO:

- Scheduler (24/7):    python main.py --mode scheduler
- Uma vez (teste):      python main.py --mode once
- Dry run (sem posts):  python main.py --mode dry-run
- Dashboard (stats):    python main.py --mode dashboard
- Health check:         python main.py --mode health

CREDENCIAIS NECESSÁRIAS:

1. PostgreSQL - Database URL
2. Telegram - Bot token, channel/group IDs
3. Twilio - Account SID, Auth token (WhatsApp)
4. Amazon - Access key, secret key, partner tag

Todas disponíveis gratuitamente ou com custo mínimo.

SETUP RÁPIDO (VPS):

1. Provision Ubuntu 22.04 droplet/instance
2. SSH e execute:
   - apt update && apt upgrade
   - apt install python3-pip postgresql
3. Clone project: git clone <url>
4. bash setup.sh
5. Configure .env
6. Copie brasildeals.service para /etc/systemd/system/
7. systemctl enable brasildeals && systemctl start brasildeals

Ver DEPLOYMENT.md para guia completo.

GERAÇÃO DE RECEITA
================================================================================

MODELO:

1. Scrape deals de RSS feeds
2. Distribua para audience em Telegram/WhatsApp
3. Usuarios clicam em links de afiliado Amazon
4. Você recebe comissão 4% (padrão Amazon Associates)

EXEMPLO:

- 50 deals/dia
- 100 subscribers
- 5% click rate = 5 cliques/dia
- $50 preço médio
- 4% comissão = $2 por venda
- 1-2 vendas/dia = $50-100/dia = $1500-3000/mês

POTENCIAL:

Com promoção ativa:
- $500-5000/mês com 1-2 horas de trabalho/dia
- Escalável a $20,000+/mês com multiple channels e produtos
