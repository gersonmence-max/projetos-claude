# LandHQ Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Monitor tax deed auctions across 50 US counties, enrich parcels with public risk/market data, score them 0-100, and serve a Portuguese-language dashboard.

**Architecture:** 3-layer pipeline (auto-filters → math scoring → Anthropic AI fine analysis), daily APScheduler job, FastAPI backend, Next.js 14 frontend.

**Tech Stack:** Python 3.11, Playwright, httpx, PostgreSQL/Supabase, FastAPI, Next.js 14 + TypeScript + Tailwind, APScheduler, Anthropic API, Resend, Railway.
