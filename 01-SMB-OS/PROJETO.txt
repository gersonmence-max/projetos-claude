=============================================================================
PROJETO: SMB OS (Small Business Operating System)
=============================================================================

STATUS: Em planejamento / Parcialmente desenvolvido
PRIORIDADE: ALTA
CATEGORIA: SaaS / ERP
MERCADO-ALVO: Pequenas empresas brasileiras (faturamento até R$ 1,5M/mês)

-----------------------------------------------------------------------------
O QUE É
-----------------------------------------------------------------------------

SaaS completo de gestão empresarial voltado para pequenas empresas, com foco
inicial no segmento de joalherias. Plataforma multi-tenant que integra CRM,
vendas, estoque, faturamento (NF-e/NFS-e), financeiro, ordens de serviço e
IA conversacional nativa.

Diferencial competitivo: IA integrada em todos os módulos + foco em 
terceirização de ourives (gap que Omie/Bling não cobrem).

-----------------------------------------------------------------------------
STACK TÉCNICA
-----------------------------------------------------------------------------

Backend:  FastAPI + Python
Frontend: Next.js + TypeScript + TailwindCSS
Mobile:   Expo (React Native)
Banco:    PostgreSQL (Supabase) com Row-Level Security (multi-tenant)
Infra:    Railway (backend) + Vercel (frontend)
Payments: Stripe + Plaid

-----------------------------------------------------------------------------
MODELO DE PREÇOS
-----------------------------------------------------------------------------

Starter:    R$ 397/mês (setup R$ 497)
Pro:        R$ 697/mês (setup R$ 997)
Enterprise: R$ 1.197/mês (setup R$ 1.997)

-----------------------------------------------------------------------------
ROADMAP DE DESENVOLVIMENTO (8 SESSÕES)
-----------------------------------------------------------------------------

Sessão 1: Setup monorepo + Supabase + autenticação multi-tenant
Sessão 2: CRM - modelo Cliente + CRUD + busca
Sessão 3: Ordem de Serviço (OS) de restauro + terceirização ourives
Sessão 4: PDV básico + NF-e/NFS-e
Sessão 5: Financeiro básico + DRE + fluxo de caixa
Sessão 6: Estoque por composição (ouro/pedras)
Sessão 7: Portal do cliente + fotos antes/depois
Sessão 8: App mobile Expo
