PROJETO: DAILY MUSIC GENERATION - SOUNDON
=============================================================================

Data de criação: Maio de 2026
Descrição: Automação diária de geração e distribuição de música

O QUÊ:
Sistema totalmente automatizado que:
1. Gera 1 música nova TODOS OS DIAS (9:00 da manhã)
2. Escolhe automaticamente a melhor versão
3. Faz upload no SoundOn
4. Distribui para todas as plataformas:
   - Spotify
   - Apple Music
   - YouTube Music
   - Amazon Music
   - E mais 15+ plataformas via SoundOn

GÊNERO:
🎵 Eletrônico Dançante (estilo Alok)
- Batidas energéticas
- 120-130 BPM
- Instrumentação moderna

STACK:
- Python 3.9+
- Suno API (geração de música IA)
- SoundOn API (distribuição)
- Scheduled Tasks (cron/systemd)
- Google Drive (backup automático)

FLUXO DE AUTOMAÇÃO:
9:00 AM → Trigger diário
   ↓
Suno API gera 2 versões da música
   ↓
Valida qualidade (duração, formato)
   ↓
Seleciona melhor versão automaticamente
   ↓
Upload para SoundOn
   ↓
SoundOn distribui para 15+ plataformas
   ↓
24-48h: Música ao vivo em Spotify, Apple Music, etc
   ↓
Backup em Google Drive + logs
