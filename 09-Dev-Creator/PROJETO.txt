Descrição: Sistema Autônomo de Desenvolvimento

O QUÊ:
Sistema que automatiza a criação de código/projetos a partir de prompts em linguagem natural.
Você descreve o que quer, e o sistema:
1. Gera o código automaticamente
2. Executa testes
3. Corrige erros automaticamente
4. Valida a qualidade

STACK:
- Python 3.9+
- Claude API (via executor.py)
- Modular architecture (generator, executor, validator, fixer)
- Web interface (server.py)

STATUS:
✅ MVP funcional
- Generator: ✅ Cria código automaticamente
- Executor: ✅ Executa e testa
- Fixer: ✅ Corrige erros
- Validator: ✅ Valida qualidade
- Reporter: ✅ Gera relatórios
